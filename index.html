UbicaFoto

Ubica tus fotos, compártelas y expórtalas a Google Earth.

Selecciona el archivo original de la carpeta DCIM/Camera. Puedes cambiar el nombre, agregar una nota y decidir si el KMZ incluye las fotos.
🔒 Las fotografías se procesan dentro de este dispositivo. La app no las envía a un servidor.
🛡️ El gesto de deslizar hacia abajo para actualizar está bloqueado mientras usas la app. Para borrar el trabajo actual, utiliza Nuevo trabajo.
